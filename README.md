## Detroit ShotSpotter analysis (2021-2026)

**Ben Warren, Detroit News**

In this repository, I analyze data from the Detroit ShotSpotter gunshot detection system. I compare it to 911 calls, real-time crime data and geographic variables.

------------------------

**Files:**
- `bopc_comparison` (analyze differences between Board of Police Commissioners data and ShotSpotter data)
- `shotspotter_2021_2026` (analyze ShotSpotter data linked to crime incidents, firearm seizures and arrests)
- `shotspotter_sca_analysis` (compares scout car areas in ShotSpotter coverage area vs. outside of coverage area in 2024-2025)
